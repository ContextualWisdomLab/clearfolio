## 2025-03-09 - [콘텐츠 해시 16진수 변환 최적화]
**Learning:** 파일 업로드 시 콘텐츠 해시를 생성할 때 16진수 변환을 위해 루프 내에서 `String.format("%02x", b)`를 사용하는 것은 정규식 처리와 반복적인 문자열 할당으로 인해 성능에 큰 병목이 됩니다. Java 17부터 추가된 `java.util.HexFormat`을 사용하면 메모리 할당을 줄이고 실행 시간을 대폭 단축할 수 있습니다.
**Action:** `contentHash` 메서드를 비롯해 코드베이스 전반에서 16진수 변환 루프를 찾아 `HexFormat.of().formatHex(bytes)`로 교체합니다.
