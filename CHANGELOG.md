# 변경 로그 (Changelog)

## [Unreleased]

### UX 및 접근성 개선 (UX & Accessibility Improvements)
* 외부 링크 및 JSON 부트스트랩 열기 시 새 탭에서 열리도록 `target="_blank"`, `rel="noopener noreferrer"` 속성을 추가하여 사용자가 문서 뷰어 컨텍스트를 유지할 수 있도록 개선했습니다.
* 스크린 리더 사용자를 위해 새 탭에서 열림을 명확히 안내하는 `aria-label` 속성을 외부 링크 요소에 추가하여 웹 접근성을 향상시켰습니다.
